Custom Field Codegen

Generate ACF field code from Figma layers

Fields Preview (Drag to reorder)

Select layers and click "Generate Code"

v1.0.0

📘 How to Use

1
Create Layers in Figma
Create text layers or frames with names that describe your ACF fields. Add prefixes to automatically set field types.
2
Select Layers
Click layers while holding Shift to select multiple layers.
3
Configure Settings
Set the field group name and choose where it should appear.
4
Generate & Copy
Click "Generate Code", review fields, then click "Copy Code". Paste into your WordPress theme's functions.php.

⚠️ Layer Name Requirements

Layer names must contain at least one alphanumeric character or Japanese character.

✓ Valid:
title, 123, img_hero, 画像_, テスト

✗ Invalid:
@@@@@ (symbols only)
😎 (emoji only)
### (special characters only)

Tip: If you want to use symbols or emoji, add a prefix or alphanumeric characters.

⚙️ Requirements

WordPress: 5.0 or higher
Plugin: Advanced Custom Fields (Free or Pro)
PHP: 7.4 or higher
